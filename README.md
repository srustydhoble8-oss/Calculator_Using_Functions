# 🧮 Calculator Using Functions

A simple calculator built with Python using functions.

## ✨ Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🔢 Modulus
* ⚡ Power Calculation

## 📚 Concepts Used

* 🐍 Python Functions (`def`)
* 🔄 While Loop
* 🎯 Conditional Statements
* ⌨️ User Input
* 🔢 Mathematical Operations
* ↩️ Return Statements

## 🎮 How It Works

1. User selects an operation.
2. User enters two numbers.
3. The selected function is called.
4. The result is displayed.

## 🛠 Functions Included

* `add(a, b)`
* `sub(a, b)`
* `mult(a, b)`
* `div(a, b)`
* `modulus(a, b)`
* `power(a, b)`

## 💻 Example

Enter your choice: 1

Enter number a: 10

Enter number b: 5

Result: 15
